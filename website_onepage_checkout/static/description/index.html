Website Onepage Checkout

Streamline your eCommerce checkout by consolidating all steps into a single page with interactive accordion panels. Reduce cart abandonment and improve conversion rates.

Key Features

Single Page Checkout

All checkout steps on one page with accordion panels.

Time Saving

Faster checkout experience with fewer page redirects.

Customizable Panels

Enable/disable panels and customize labels per your needs.

Configurable Fields

Set required address fields for billing and shipping.

Secure Payment

Integrated payment processing with all Odoo providers.

Enable the OnePage Checkout

Activate the module from Website Webkul Addons settings and click Configuration.

One Page Check Out Configuration

Configure panel visibility, custom labels, required address fields, and website-specific settings from a single form.

Check Out Page - Billing Information

The first panel opens with billing address selection. Select an existing address or continue to the next step.

Check Out Page - Shipping Information

Add or select a shipping address. Completed panels turn green and remain accessible.

Check Out Page - Delivery Method

Choose from available delivery methods. Rates are calculated in real-time based on cart value and shipping address.

Check Out Page - Order Preview & Payment

Review your cart, adjust quantities, apply discount codes, and complete payment - all without leaving the page.

Payment Confirmation

Order confirmation with payment details, product summary, and shipping information.

How It Works

Step Panel Description
1 Billing Information Select billing address, click Continue
2 Shipping Information Add or select shipping address
3 Delivery Method Choose delivery carrier with live rates
4 Order Preview & Payment Review cart, adjust qty, apply discount, pay

Completed panels turn green and can be revisited anytime.

Pending panels stay purple until reached via Continue.
